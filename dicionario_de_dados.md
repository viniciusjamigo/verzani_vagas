# Dicionário de Dados - vagas_ori_BRUNA.csv

Este arquivo descreve as colunas encontradas no arquivo `vagas_ori_BRUNA.csv`, que serve como base de dados para o dashboard de recrutamento.

| Nome da Coluna | Tipo de Dado | Descrição | Exemplo |
| :--- | :--- | :--- | :--- |
| `STATUS` | Texto | Anotações internas sobre o status do candidato ou do processo. | `ASO PENDENTE` |
| `RECRUTADOR` | Texto | Nome do recrutador responsável pela vaga. | `BRUNA SOUZA` |
| `NOME` | Texto | Nome do candidato que está no processo para a vaga. | `Vanilza Fonseca` |
| `PREVISÃO INÍCIO` | Texto / Data | Pode conter o nome do candidato novamente ou a data de previsão para o início. | `Vanilza Fonseca` |
| `STATUS DO PROCESO` | Texto | Etapa atual do processo seletivo para a vaga. | `EM ADMISSÃO` |
| `Empresa` | Texto | Empresa ou unidade de negócio que está contratando. | `SERV` |
| `COO`, `Executivo`, ... | Texto | Nomes dos gestores em diferentes níveis hierárquicos. | `MARCOS.MORETTI` |
| `Grupo Econômico` | Texto | O grupo empresarial ao qual a vaga pertence. | `DROGARIA ARAUJO` |
| `Grupo Econômico com...` | Texto | Combinação do gestor de menor nível com o grupo. | `KELLY.BATISTA - ALLOS` |
| `Contrato` | Texto | Código ou identificador do contrato de serviço. | `400106035-BOULEVARD...` |
| `UF da OI`, `Cidade da OI`,... | Texto | Informações detalhadas de localização da vaga. | `MG`, `BELO HORIZONTE` |
| `Código da Requisição` | Texto/Número | Identificador único para a requisição de vaga. | `264209` |
| `Código da Vaga` | Texto/Número | Identificador único para a vaga. | `18974/25` |
| `IDPV` | Texto/Número | Outro identificador para a vaga (ID Ponto de Venda?). | `461369` |
| `Hub de Vagas` | Número | Identificador relacionado ao sistema de vagas. | `1` |
| `Título do Cargo` | Texto | O cargo ou função da vaga. | `AGENTE ASSEIO CONSERVACAO`|
| `Descrição do Sexo` | Texto | Requisito de gênero para a vaga. | `Indiferente` |
| `Descrição da Escala` | Texto | Descrição do horário e dias de trabalho. | `5x1 14:00 as 22:20 - 1:00` |
| `Salário` | Numérico | Valor do salário da vaga. | `1649,12` |
| `Status da Vaga Agrupadas`| Texto | Uma categoria mais ampla do status da vaga. | `Finalizada` |
| `Status da Vaga` | Texto | O status detalhado da vaga, usado nos gráficos. | `Finalizado - Vaga Preenchida` |
| `Situação Vagas` | Texto | Indica se a vaga está dentro ou fora do SLA (Acordo de Nível de Serviço). | `Fora do SLA` |
| `Dias em Aberto` | Numérico | Número de dias que a vaga está aberta. | `14` |
| `Qtd em Andamento` | Numérico | Quantidade de vagas em andamento (geralmente 1 para a linha). | `1` |
| `Qtd Finalizadas` | Numérico | Quantidade de vagas finalizadas (geralmente 1 para a linha). | `1` |
| `Recrutamento e Seleção` | Data | Data de início do processo de R&S. | `12/08/2025` |
| `Finalizada` | Data | Data em que a vaga foi finalizada/preenchida. | `26/08/2025` |
| `Descrição do Motivo` | Texto | Motivo da abertura da vaga. | `Aumento De Quadro` |
| `Anotações` | Texto | Campo de texto livre, frequentemente usado para listar benefícios. | `Benefícios: vale: Va...` |
| `Habilitação Técnica` | Texto | Requisitos de habilitação técnica (ex: CNH). | |
| `Data de Início da OI` | Data | Data de início da Ordem de Início do serviço. | `01/02/2019` |
| `Abrangência Vagas` | Texto | Região de abrangência da vaga. | `RJ/MG/CO` |
